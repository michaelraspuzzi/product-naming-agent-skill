# Naming Science (first principles)

Knowledge base behind the `name-phonetics`, `name-linguistics`, and `name-psychology` evaluation agents.

## 1. Sound symbolism / phonetics
- **Sapir 1929 (mil/mal):** /i/ "ee" (high front) → small/fast/light/sharp; /a,o,u/ (low/back) → large/heavy/premium. ~80% agreement.
- **Bouba/kiki (Köhler 1929; Ramachandran & Hubbard 2001):** 95-98% match kiki (plosives+front vowels)→spiky/fast; bouba (sonorants+round vowels)→soft/calm. Cross-cultural.
- **Map:** V=energy/velocity · B/T=reliability · X/Z=fast/modern · plosives(b,p,t,k,d,g)=impact/techy · fricatives(f,s,v,z,sh)=light/airy/fast · sonorants(l,m,n,r,w)=soft/flowing/calm · alliteration/repetition=+memorability.
- **Rule:** fast/tech/sharp → plosives+front vowels; soft/wellness/premium → sonorants+back vowels. Flag sound-vs-positioning mismatch.

## 2. Linguistics / morphology
- **Construction:** real word · compound (FedEx) · blend (Pinterest, Groupon) · affixation (-ly,-ify,-o/-a/-ia, latin/greek roots) · misspelling (Lyft, Flickr) · coined (Kodak, Xerox) · foreign word.
- **Length:** 1-2 syllables best, 3 ok, 4+ penalize. Favor CV alternation; avoid stalling clusters.
- **Spell-able:** correct spelling after hearing once (radio test). **Say-able:** one obvious pronunciation. Misspellings/coinages buy availability but cost spell-ability.
- **Blends:** reward shared phoneme at the seam (Gro-upon).

## 3. Psychology
- **Cognitive fluency (Alter & Oppenheimer 2006, PNAS):** easy-to-process names are trusted/liked/valued more; fluent stock names outperformed; pronounceable tickers (KAR) beat (RDO).
- **Von Restorff 1933:** the distinctive item is remembered far better → reward standout vs competitors.
- **Concreteness & emotion:** picture-evoking, feeling-tied names recalled better.
- **Connotation/priming:** audit substrings/homophones/associations for negatives.
- **Aging:** descriptive = instant clarity but boxes you in (Pets.com); suggestive/abstract = empty at first but scales (Amazon, Uber). More expansion ambition → reward abstract.

## 4. Naming taxonomy + trademark distinctiveness
Generic (unregistrable) < Descriptive (weak TM, limits growth) < Suggestive/Evocative (best balance) < Arbitrary (Apple) / Fanciful-Invented (Kodak, strongest TM). Also: Founder/eponymous, Acronym, Geographic (weaker TM). **Default recommendation: Suggestive/Evocative.**

## 5. Practical screening
- **Trademark first** — it can kill a name. DuPont factors; dominant two: **mark similarity** (sight/sound/meaning) × **goods relatedness**. Nice classes: goods 1-34, services 35-45. Same/similar mark in a related class collides.
- **Domain:** exact .com preferred; then .io/.ai/.co for tech. Dictionary-word .coms are taken.
- **Social handles** across platforms that matter.
- **Cross-language:** screen for vulgar/negative meaning & bad homophones in target-market languages. The Chevy "Nova/no va" story is a **debunked myth** — don't trust naming folklore; only flag real, verifiable issues.
- **Final gate:** radio test + single-pronunciation test + syllable/rhythm check.

### Screening checklist
```
[ ] Not generic/merely descriptive; ideally suggestive/arbitrary/fanciful
[ ] No identical/confusingly-similar mark in related Nice class(es)
[ ] .com (or acceptable TLD) + key social handles
[ ] No vulgar/negative meaning or bad homophone in target languages
[ ] One obvious pronunciation; spell-able from hearing once
[ ] 1-3 syllables, clean rhythm, no stalling clusters
[ ] Sound profile matches positioning
[ ] Distinctive vs named competitors
[ ] Still fits if product range triples (abstract > descriptive for expansion)
```

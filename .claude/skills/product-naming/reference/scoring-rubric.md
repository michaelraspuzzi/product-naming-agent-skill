# Scoring Rubric & Composite Score

How the orchestrator combines the evaluation agents in Phase 3.

## Composite score (0-15, +bonuses)
| Lens | Agent | Range |
|---|---|---|
| Sound-fit | `name-phonetics` | 0-5 |
| Usability (construction/say/spell/distinctiveness) | `name-linguistics` | 0-5 |
| Psychology (fluency/distinctive/connotation/aging) | `name-psychology` | 0-5 |
| **Base total** | | **0-15** |

Bonuses / penalties:
- **+1 cross-school agreement** — ≥3 firm agents independently produced the same/similar name. Strong signal.
- **+1 strategy fit** — the orchestrator judges it nails the single original idea from the brief.
- **−∞ hard flags** — generic (unregistrable), sound contradicts positioning, negative connotation, fatal cross-language meaning, same-category competitor clash. Drop unless exceptional + justified.

## Tiebreakers (Igor's A.S.S. count)
When two finalists tie: count cultural associations/idioms each carries; the richer one wins. If still tied, sketch a tagline off each — the one that yields the better line wins.

## Essence-coverage gate (required)
Before advancing the shortlist, confirm the obvious on-the-nose name(s) from `namer-essence` — the core category verb and feeling-word (e.g., for a cert tutor: *Certify, Pass, Prove, Certain*) — are either present in the shortlist or were *consciously* rejected with a stated reason. The literal baseline is the floor every distinctive candidate must beat; it must never be skipped by accident, only out-competed on purpose.

## The "comfortable = wrong" check
Before finalizing #1, ask: is the whole shortlist *safe*? Lexicon's rule — internal comfort signals you don't have the right name yet. Always surface the boldest/most polarizing candidate and argue why it might be the right choice, even if it doesn't top the composite score.

## Output thresholds
- Need ≥3 names scoring ≥10/15 that survive availability to deliver a top-5. If not, loop back to Phase 2 with a sharpened brief rather than shipping weak names.
